Helpful hints

How does this site work?

This system uses OpenAI's GPT-4 to give responses to user queries.

How is this site different from ChatGPT?

This system contains a library of pre-formatted excerpts from the relevant regulations. Behind the scenes, the system picks one or more relevant policies from among that library, appends it to your query, and then directs GPT-4 to refer to that particular text when formulating its response. The result should usually be more focused and accurate than you would get if you put the same query into ChatGPT.

Which sets of regulations are in the library?

Right now, the library contains portions of the Foreign Affairs Manual. In the future it will contain other related sets of regulations, such as the FAH, DSSR, ADS, AIDAR, etc.

Are the responses this system gives completely accurate?

Not always! The responses should usually cite specific regulations, which you can use to double-check their accuracy.

Should I include any Personally Identifiable Information (PII) in the questions I submit?

No.
